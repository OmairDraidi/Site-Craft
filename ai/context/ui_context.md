# ui_context.md — SiteCraft UI/UX AI Context

## 1. Purpose
To guide AI in generating all UI/UX elements consistently across SiteCraft using:
- Brand identity
- Color system
- Typography rules
- Spacing system
- Component library rules
- Figma Make prompt templates

---

## 2. Brand Identity

### Colors
- Gold primary: #F6C453
- Black gradient: #0A0A0A → #121212 → #1A1A1A
- Gray scale: gray-200 → gray-900

### Fonts
- Poppins (titles)
- Inter (body)

### Effects
- Shadows: Gold glow + Dark elevation
- Rounded corners: 12–16px

### Spacing
Scale:
4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 / 96

---

## 3. UI Components Rules

### Buttons
- Primary = gold
- Secondary = outline gold
- Ghost = transparent
- States: default / hover / pressed / disabled

### Inputs
- Dark input
- Gold focus border
- Error state = red border

### Cards
- Dark backgrounds
- 16px padding
- Gold icon headers

### Sidebar
- Black background
- Gold active indicator
- White text

### Tables
- Dark rows
- Hover effect
- Status pills

---

## 4. Figma Make Prompt Rules
AI must generate prompts with:

- Clear layout structure  
- Color and typography alignment  
- Proper auto-layout  
- Page sections defined  
- Output expectations  

Figma prompts must include:
- Container widths
- Grid systems
- Section headers
- Component variants
- Dark mode only
