# SiteCraft — Project Overview (project.md)

## 1. Vision
SiteCraft is a multi-tenant SaaS platform that allows users to instantly create professional websites using ready-made templates and an AI‑assisted builder.

## 2. Core Pillars
- Website Builder (drag & drop)
- Template Library
- Domain Management
- Multi-Tenant System
- Admin Dashboard
- Public Sites Rendering
- Billing & Subscription Plans

## 3. Target Users
- Freelancers
- Small businesses
- Creators & entrepreneurs
- Agencies building multiple sites

## 4. Key Features
- Subdomain per tenant
- Custom domain support
- Template marketplace
- Real-time publishing
- Dashboard analytics
- AI-powered UI generation
- Multi-user roles (Owner, Admin, Editor)

## 5. Architecture Summary
Frontend:
- React 19, Tailwind, Zustand, React Query, Shadcn

Backend:
- ASP.NET Core 8 + MySQL + Redis + Hangfire + S3

Documents:
- PRD
- System Analysis
- Wireframes
- UI Prompts
- Architecture
- Conventions
- AI contexts
